# Pizzaria Nostra - Web Application

Interface moderna e responsiva desenvolvida para a Pizzaria Nostra, focada em alta performance e experiência do usuário (UX). O projeto apresenta uma solução completa para visualização de cardápio, gestão de pedidos e informações institucionais.

## Tecnologias Utilizadas

- **React 19**: Biblioteca para construção da interface.
- **TypeScript**: Garantia de tipagem estática e segurança de código.
- **Tailwind CSS**: Estilização baseada em utilitários para design responsivo.
- **Vite**: Ferramenta de build e desenvolvimento rápido.

## Funcionalidades principais

- **Cardápio Interativo**: Visualização detalhada de produtos com sistema de adição ao carrinho.
- **Gestão de Pedidos**: Carrinho de compras com integração direta para finalização via WhatsApp.
- **Navegação Inteligente**: Sistema de scroll suave com compensação de offset para menus fixos.
- **Área do Usuário**: Interfaces de autenticação (Login/Cadastro) com validações básicas.
- **Institucional**: Seções sobre a filosofia da marca, depoimentos de clientes e localização via Google Maps.

## Estrutura do Projeto

O projeto segue uma organização modular:

- `App.tsx`: Gerenciador de estado global e roteamento da landing page.
- `components/`: Componentes modulares e reutilizáveis (Navbar, Hero, Menu, Cart, etc).
- `constants.tsx`: Centralização de dados estáticos e configurações de contato.
- `types.ts`: Definições de interfaces para consistência de dados.
